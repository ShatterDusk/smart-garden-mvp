# 数据库日志服务 - 一期计划

> 目标：实现日志文件获取功能，支持在根目录获取前后端日志用于测试分析

---

## 一、项目背景

当前日志分散在前后端各自存储，需要统一接口获取日志文件进行分析，为后续数据库日志方案设计提供数据支撑。

---

## 二、目标范围

### 2.1 核心目标
- 通过 API 接口获取后端日志文件（`backend/server/logs/`）
- 通过 API 接口获取前端日志文件（`frontend/logs/`）
- API 访问需验证密钥

### 2.2 非目标
- 不实现数据库日志存储（二期内容）
- 不修改现有日志产生逻辑
- 不实现日志实时推送

---

## 三、技术方案

### 3.1 后端 API 设计

#### 密钥验证方式
- Query 参数：`?accessKey=xxx`
- 密钥配置：环境变量 `LOG_ACCESS_KEY`

#### 接口列表

| 接口 | 方法 | 功能 | 参数 |
|------|------|------|------|
| `/api/logs/files` | GET | 获取日志文件列表 | `accessKey`, `source` (backend/frontend) |
| `/api/logs/content` | GET | 获取指定日志内容 | `accessKey`, `file`, `lines`, `source` |
| `/api/logs/search` | GET | 搜索日志内容 | `accessKey`, `file`, `keyword`, `source` |
| `/api/logs/clear` | DELETE | 清空指定日志 | `accessKey`, `file`, `source` |

#### 响应格式
```json
{
  "code": 0,
  "message": "success",
  "data": { ... }
}
```

### 3.2 前端 API 封装

- 新建文件：`frontend/utils/logApi.js`
- 不使用 `config.API_BASE_URL`，直接使用完整本地地址
- 基础地址：`http://localhost:3000/api/logs`

```javascript
const LOG_API_BASE = 'http://localhost:3000/api/logs';
const ACCESS_KEY = 'your-local-dev-key';

// 获取后端日志列表
getBackendLogs: () => {
  return new Promise((resolve, reject) => {
    wx.request({
      url: `${LOG_API_BASE}/files?accessKey=${ACCESS_KEY}&source=backend`,
      success: resolve,
      fail: reject
    });
  });
}
```

## 四、实施任务

### 任务 1：后端密钥验证中间件
**文件**：`backend/server/src/middleware/logAuth.js`

```javascript
/**
 * 日志接口密钥验证中间件
 */
const LOG_ACCESS_KEY = process.env.LOG_ACCESS_KEY || 'dev-log-key-2024';

module.exports = (req, res, next) => {
  const key = req.query.accessKey;
  if (!key || key !== LOG_ACCESS_KEY) {
    return res.status(403).json({
      code: 403,
      message: '无效的访问密钥',
      data: null
    });
  }
  next();
};
```

### 任务 4：前端日志 API 封装
**文件**：`frontend/utils/logApi.js`

```javascript
/**
 * 日志 API 封装
 * 用于获取前后端日志文件
 */
const LOG_API_BASE = 'http://localhost:3000/api/logs';
const ACCESS_KEY = 'dev-log-key-2024';

module.exports = {
  // 获取后端日志文件列表
  getBackendLogFiles: () => request(`${LOG_API_BASE}/files?accessKey=${ACCESS_KEY}&source=backend`),
  
  // 获取前端日志文件列表
  getFrontendLogFiles: () => request(`${LOG_API_BASE}/files?accessKey=${ACCESS_KEY}&source=frontend`),
  
  // 获取后端日志内容
  getBackendLogContent: (file, lines = 100) => 
    request(`${LOG_API_BASE}/content?accessKey=${ACCESS_KEY}&source=backend&file=${file}&lines=${lines}`),
  
  // 获取前端日志内容
  getFrontendLogContent: (file, lines = 100) => 
    request(`${LOG_API_BASE}/content?accessKey=${ACCESS_KEY}&source=frontend&file=${file}&lines=${lines}`),
  
  // 搜索后端日志
  searchBackendLogs: (file, keyword) => 
    request(`${LOG_API_BASE}/search?accessKey=${ACCESS_KEY}&source=backend&file=${file}&keyword=${keyword}`),
  
  // 搜索前端日志
  searchFrontendLogs: (file, keyword) => 
    request(`${LOG_API_BASE}/search?accessKey=${ACCESS_KEY}&source=frontend&file=${file}&keyword=${keyword}`),
};

function request(url) {
  return new Promise((resolve, reject) => {
    wx.request({
      url,
      method: 'GET',
      success: (res) => {
        if (res.data.code === 0) {
          resolve(res.data.data);
        } else {
          reject(new Error(res.data.message));
        }
      },
      fail: reject
    });
  });
}
```
