**创建日期**: 2026-04-07  
**适用环境**: 本地开发环境

## 一、前提条件

### 1.1 安装 MySQL

**Windows 安装**:
1. 下载 MySQL Installer: https://dev.mysql.com/downloads/installer/
2. 选择 "Server only" 或 "Full" 安装
3. 记住设置的 root 密码

**或使用 XAMPP** (推荐，更简单):
1. 下载 XAMPP: https://www.apachefriends.org/
2. 安装时选择 MySQL
3. 启动 XAMPP Control Panel
4. 点击 MySQL 的 "Start" 按钮

**或使用 Docker**:
```bash
docker run -d --name mysql-dev \
  -e MYSQL_ROOT_PASSWORD=root \
  -e MYSQL_DATABASE=smart_garden_dev \
  -p 3306:3306 \
  mysql:8.0
```

### 1.2 验证 MySQL 安装

```bash
# 如果 mysql 命令可用
mysql --version

# 或使用 XAMPP 的 mysql
C:\xampp\mysql\bin\mysql --version
```

## 三、后端配置

### 3.1 环境变量文件

已创建 `backend/server/.env.local`:

```env
# 本地开发环境配置
NODE_ENV=development

# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_NAME=smart_garden_dev
DB_USER=root
DB_PASSWORD=你的密码
DB_DIALECT=mysql
DB_SSL=false
DB_LOGGING=true

# JWT配置
JWT_SECRET=local_dev_jwt_secret_key_not_for_production

# 其他配置...
```

### 3.2 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| DB_HOST | 数据库主机 | localhost |
| DB_PORT | 数据库端口 | 3306 |
| DB_NAME | 数据库名称 | smart_garden_dev |
| DB_USER | 数据库用户 | root |
| DB_PASSWORD | 数据库密码 | 空 |
| DB_LOGGING | 是否打印SQL日志 | true |

### 3.3 修改密码

如果 MySQL root 用户有密码，修改 `.env.local`:
```env
DB_PASSWORD=你的实际密码
```

## 五、常见问题

### 5.1 数据库连接失败

**问题**: `Access denied for user 'root'@'localhost'`

**解决方案**:
1. 检查密码是否正确
2. 创建新用户:
```sql
CREATE USER 'dev_user'@'localhost' IDENTIFIED BY 'dev_password';
GRANT ALL PRIVILEGES ON smart_garden_dev.* TO 'dev_user'@'localhost';
FLUSH PRIVILEGES;
```
3. 更新 `.env.local` 使用新用户

### 5.2 端口被占用

**问题**: `Port 3306 is already in use`

**解决方案**:
1. 查找占用端口的进程:
```bash
netstat -ano | findstr :3306
```
2. 结束进程或修改 MySQL 端口
3. 更新 `.env.local` 中的 `DB_PORT`

### 5.3 字符集问题

**问题**: 中文显示乱码

**解决方案**:
确保数据库使用 utf8mb4:
```sql
ALTER DATABASE smart_garden_dev 
  CHARACTER SET utf8mb4 
  COLLATE utf8mb4_unicode_ci;
```
