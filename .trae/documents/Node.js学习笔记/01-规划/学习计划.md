### 阶段 3：异步编程（第 6-8 天）⭐ 重点

**目标**：彻底理解 Promise 和 async/await

| 时间 | 内容 | 产出 | 状态 |
|:---|:---|:---|:---:|
| 第 6 天 | 同步 vs 异步、Callback 模式 | 理解异步概念 | ⬜ |
| 第 7 天 | Promise 详解、then/catch/finally | 能写 Promise | ⬜ |
| 第 8 天 | async/await、try-catch、并行执行 | 熟练使用 async | ⬜ |

**关键代码**：
```javascript
// BaseService.js 中的异步操作
async findById(id) {
  try {
    return await this.model.findByPk(id);
  } catch (err) {
    logger.error(`${this.modelName}.findById error:`, err);
    throw err;
  }
}

// 并行执行
const [user, posts] = await Promise.all([
  getUser(),
  getPosts()
]);
```

**练习任务**：
- [ ] 把 Callback 改成 Promise
- [ ] 用 async/await 重写 Promise 链
- [ ] 实现并行请求数据
- [ ] 处理异步错误
