## 设计原则

1. **高内聚低耦合** - 每个模块职责单一
2. **可测试性** - 支持单元测试和集成测试
3. **可扩展性** - 插件化架构
4. **可配置性** - 所有参数可配置
5. **可观测性** - 完善的日志和监控

---

## 核心特性

### 三层数据生成
- Layer 1: 自然模拟（物理规律）
- Layer 2: 场景约束（范围限制）
- Layer 3: 事件驱动（用户编排）

### 事件时间线
- 可编排未来事件
- 虚拟时间加速
- 时间旅行调试

### 多设备支持
- 设备实例隔离
- 批量操作
- 统一监控
