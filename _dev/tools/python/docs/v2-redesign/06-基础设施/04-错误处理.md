# 错误处理

## 1. 概述

错误处理系统提供统一的异常捕获、处理和恢复机制，确保系统稳定性和用户体验。

## 2. 错误分类

| 类别 | 说明 | 示例 |
|------|------|------|
| 业务错误 | 业务逻辑错误 | 设备不存在、场景切换失败 |
| 系统错误 | 系统级错误 | 内存不足、文件读写失败 |
| 网络错误 | 网络通信错误 | 连接超时、DNS解析失败 |
| 配置错误 | 配置相关错误 | 配置格式错误、缺少必填项 |
| 验证错误 | 数据验证错误 | 参数范围错误、类型不匹配 |

## 4. 异常类设计

```python
class VirtualDeviceError(Exception):
    """基础异常类"""
    
    def __init__(
        self,
        code: ErrorCode,
        message: str,
        details: dict = None,
        cause: Exception = None
    ):
        self.code = code
        self.message = message
        self.details = details or {}
        self.cause = cause
        super().__init__(message)
    
    def to_dict(self) -> dict:
        """转换为字典"""
        return {
            "code": self.code.value,
            "message": self.message,
            "details": self.details
        }

class DeviceError(VirtualDeviceError):
    """设备相关错误"""
    pass

class ScenarioError(VirtualDeviceError):
    """场景相关错误"""
    pass

class TimelineError(VirtualDeviceError):
    """时间线相关错误"""
    pass
```

## 5. 错误处理器

```python
import functools
import traceback

class ErrorHandler:
    """错误处理器"""
    
    def __init__(self, logger):
        self._logger = logger
        self._handlers: Dict[ErrorCode, callable] = {}
    
    def register(self, code: ErrorCode, handler: callable):
        """注册错误处理器"""
        self._handlers[code] = handler
    
    async def handle(self, error: VirtualDeviceError) -> dict:
        """处理错误"""
        # 记录错误
        self._logger.error(
            f"Error occurred: {error.code.name}",
            code=error.code.value,
            message=error.message,
            details=error.details,
            traceback=traceback.format_exc() if error.cause else None
        )
        
        # 查找处理器
        handler = self._handlers.get(error.code)
        if handler:
            return await handler(error)
        
        # 默认处理
        return error.to_dict()

def with_error_handler(handler: ErrorHandler):
    """错误处理装饰器"""
    def decorator(func):
        @functools.wraps(func)
        async def wrapper(*args, **kwargs):
            try:
                return await func(*args, **kwargs)
            except VirtualDeviceError as e:
                return await handler.handle(e)
            except Exception as e:
                # 包装未知错误
                error = VirtualDeviceError(
                    code=ErrorCode.UNKNOWN_ERROR,
                    message=str(e),
                    cause=e
                )
                return await handler.handle(error)
        return wrapper
    return decorator
```

## 6. 恢复策略

| 错误类型 | 恢复策略 |
|----------|----------|
| 临时网络错误 | 指数退避重试 |
| 设备连接断开 | 自动重连 |
| 数据生成失败 | 使用默认值 |
| 场景切换失败 | 回滚到原场景 |
| 配置错误 | 使用默认配置 |
