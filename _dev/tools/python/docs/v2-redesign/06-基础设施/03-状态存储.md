# 状态存储

## 1. 概述

状态存储系统管理虚拟设备的运行时状态，支持内存缓存和持久化存储，确保数据一致性和可靠性。

## 2. 状态类型

| 状态类型 | 存储方式 | 持久化 | 说明 |
|----------|----------|--------|------|
| 设备状态 | 内存+数据库 | 是 | 设备配置和运行时状态 |
| 传感器数据 | 内存+时序DB | 可选 | 实时和历史数据 |
| 时间线状态 | 内存 | 否 | 虚拟时间和事件队列 |
| 场景状态 | 内存 | 否 | 当前场景和过渡状态 |
| 连接状态 | 内存 | 否 | WebSocket连接信息 |

## 3. 状态管理器

```python
from typing import Dict, Any, Optional
import asyncio

class StateManager:
    """状态管理器"""
    
    def __init__(self):
        self._state: Dict[str, Any] = {}
        self._lock = asyncio.Lock()
        self._subscribers: Dict[str, list] = {}
    
    async def get(self, key: str, default: Any = None) -> Any:
        """获取状态值"""
        async with self._lock:
            return self._state.get(key, default)
    
    async def set(self, key: str, value: Any):
        """设置状态值"""
        async with self._lock:
            old_value = self._state.get(key)
            self._state[key] = value
            
            # 通知订阅者
            if key in self._subscribers:
                for callback in self._subscribers[key]:
                    await callback(key, old_value, value)
    
    async def update(self, key: str, updates: Dict[str, Any]):
        """更新嵌套状态"""
        async with self._lock:
            if key not in self._state:
                self._state[key] = {}
            
            if isinstance(self._state[key], dict):
                self._state[key].update(updates)
    
    def subscribe(self, key: str, callback):
        """订阅状态变更"""
        if key not in self._subscribers:
            self._subscribers[key] = []
        self._subscribers[key].append(callback)
    
    async def delete(self, key: str):
        """删除状态"""
        async with self._lock:
            self._state.pop(key, None)
```

## 4. 设备状态存储

```python
class DeviceStateStore:
    """设备状态存储"""
    
    def __init__(self, storage_backend):
        self._storage = storage_backend
        self._cache: Dict[str, dict] = {}
    
    async def load(self, device_id: str) -> Optional[dict]:
        """加载设备状态"""
        # 先查缓存
        if device_id in self._cache:
            return self._cache[device_id]
        
        # 再查持久化存储
        state = await self._storage.load(f"device:{device_id}")
        if state:
            self._cache[device_id] = state
        
        return state
    
    async def save(self, device_id: str, state: dict):
        """保存设备状态"""
        # 更新缓存
        self._cache[device_id] = state
        
        # 持久化存储
        await self._storage.save(f"device:{device_id}", state)
    
    async def remove(self, device_id: str):
        """删除设备状态"""
        self._cache.pop(device_id, None)
        await self._storage.delete(f"device:{device_id}")
```
