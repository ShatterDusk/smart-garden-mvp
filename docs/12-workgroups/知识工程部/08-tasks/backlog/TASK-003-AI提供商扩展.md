## 待评估问题

### 1. 是否需要实现？

**考虑因素**:
- 当前 OpenAI 和 GLM 是否满足业务需求？
- 国内 AI 提供商是否有特殊优势？（价格、性能、合规性）
- 用户是否有特定偏好？

### 2. 实现优先级

| 提供商 | 优势 | 优先级建议 |
|:---|:---|:---:|
| 百度文心一言 | 中文理解能力强 | 低 |
| 阿里通义千问 | 多模态能力强 | 低 |
| 腾讯混元 | 与微信生态整合 | 低 |

### 3. 技术可行性

需要调研各提供商的：
- API 文档和接口规范
- 图片分析能力支持
- 价格和计费方式

- 维护多个提供商增加复杂度
- 国内提供商图片分析能力可能不如 OpenAI

**行动**:
- 从配置文件中移除未实现的提供商配置
