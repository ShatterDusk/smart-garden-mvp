# 协作通知目录

## 目录说明

本目录用于存放与原工作小组的协作通知文件。

## 文件命名规范

```
{日期}-{问题编号}-修复通知.md
例如: 20260411-SA-4-004-修复通知.md

或批量通知:
{日期}-批量修复通知.md
例如: 20260411-批量修复通知.md
```
