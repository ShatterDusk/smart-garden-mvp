# 演示与分享工作组

## 简介

本工作组负责管理项目演示、技术分享、对外交流等相关材料的准备和维护工作。

## 职责范围

- 准备项目演示材料（脚本、代码片段、截图等）
- 维护技术分享内容
- 记录演示反馈和改进建议
- 沉淀演示过程中的经验和教训

## 快速开始

### 准备一次演示

1. 根据受众选择演讲材料模板
2. 准备演示资源（代码片段、截图）
3. 按照演示内容规范进行演练
4. 记录演示反馈

### 演讲材料索引

| 材料 | 适用场景 | 预计时长 |
|:---|:---|:---:|
| 产品介绍脚本 | 向非技术受众介绍产品 | 2-3分钟 |
| 架构讲解脚本 | 向技术受众讲解架构 | 3-5分钟 |
| Vibe-Coding心得 | 分享AI辅助编程经验 | 3-5分钟 |
| 教训总结案例 | 复盘项目踩坑经历 | 2-3分钟 |
| 软件设计心得 | 分享设计思想和方法论 | 2-3分钟 |
