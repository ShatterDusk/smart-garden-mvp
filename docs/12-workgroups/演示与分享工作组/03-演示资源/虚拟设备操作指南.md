## 二次开发亮点

### 1. HTTP推送支持

**文件**: `app/utils/http_helper_v2.py`

```python
class HTTPHelper:
    def __init__(self, base_url: str, device_id: str):
        self.base_url = base_url
        self.device_id = device_id
        self.session = requests.Session()
    
    def send_sensor_data(self, metrics: dict, recorded_at: datetime):
        """发送传感器数据到后端API"""
        payload = {
            "deviceId": self.device_id,
            "recordedAt": recorded_at.isoformat(),
            "metrics": metrics
        }
        response = self.session.post(
            f"{self.base_url}/api/devices/data",
            json=payload
        )
        return response.json()
```

### 2. 补传机制

**文件**: `app/core/local_task_queue.py`

```python
class LocalTaskQueue:
    """本地任务队列，支持离线存储和补传"""
    
    def __init__(self):
        self.queue = []
        self.is_uploading = False
    
    def enqueue(self, task):
        """入队，触发上传"""
        self.queue.append(task)
        if not self.is_uploading:
            self._start_upload_loop()
    
    def _start_upload_loop(self):
        """逐条上传队列中的任务"""
        self.is_uploading = True
        while self.queue:
            task = self.queue[0]
            success = self._upload_one(task)
            if success:
                self.queue.pop(0)
            else:
                break  # 失败则停止，下次重试
        self.is_uploading = False
```

### 3. 数据追赶模式

当模拟时间落后于真实时间时，自动加速追赶：

```python
def calculate_is_supplement(self, recorded_at: datetime) -> bool:
    """判断是否为补传数据"""
    now = datetime.now()
    time_diff = (now - recorded_at).total_seconds()
    return time_diff > 300  # 超过5分钟视为补传
```

## 与后端集成

### 配置 .env 文件

```env
# 后端API地址
BACKEND_URL=http://localhost:3000

# 设备认证密钥
DEVICE_SECRET=your_device_secret
```

### 数据格式转换

模拟器输出 → 后端API格式：

```json
// 模拟器输出
{
  "timestamp": "2026-04-11T10:30:00",
  "sensorName": "temperature",
  "value": 24.5
}

// 转换为后端API格式
{
  "deviceId": "DEVICE_001",
  "recordedAt": "2026-04-11T10:30:00",
  "metrics": {
    "temperature": 24.5,
    "humidity": 62.3,
    "soil_moisture": 48.7,
    "light": 12500
  }
}
```
