### 🟡 SA-7-002: sharp 实例未显式销毁（低危）

**文件位置**: `aiService.js:L360-395`

**问题描述**:
```javascript
let sharpInstance = sharp(buffer);
// ... 处理逻辑 ...
const metadata = await sharpInstance.metadata();
// ... 压缩 ...
compressedBuffer = await sharpInstance.png({ ... }).toBuffer();
// 未调用 sharpInstance.destroy()
```

**风险影响**:
- sharp 底层使用 libvips，虽然 Node.js 垃圾回收会释放内存，但显式销毁更可靠
- 高并发图片处理时可能造成内存峰值

**修复建议**:
```javascript
let sharpInstance = sharp(buffer);
try {
  const metadata = await sharpInstance.metadata();
  // ... 压缩逻辑 ...
  return compressedBuffer;
} finally {
  // 显式销毁 sharp 实例
  sharpInstance.destroy();
}
```

**优先级**: 🟡 低（当前实现无严重问题，优化项）
