### 🔍 发现的问题

#### 🟠 SA-2-001: 天气路由缺少用户认证

**严重程度**: 🟠 中  
**文件位置**: `backend/server/src/routes/weather.js:L10-19`

**问题描述**:
```javascript
// 当前代码 - 无认证中间件
router.get('/now', weatherController.getCurrentWeather);
router.get('/astronomy', weatherController.getAstronomy);
```

**风险影响**:
- 天气数据虽非敏感信息，但接口无认证可能导致滥用
- 无法追踪接口调用来源
- 可能被用于恶意刷接口

**修复建议**:
```javascript
const { authMiddleware } = require('../middleware/auth');

router.get('/now', authMiddleware, weatherController.getCurrentWeather);
router.get('/astronomy', authMiddleware, weatherController.getAstronomy);
```

**状态**: ⏳ 待修复
