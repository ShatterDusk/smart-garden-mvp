## 二、背景说明

环境数据是 AI 提供养护建议的重要依据，来源包括：
- 设备传感器数据
- 天气 API 数据
- 补偿数据（当传感器不可用时）

目前补偿逻辑分散在代码中，缺乏统一的设计文档。
