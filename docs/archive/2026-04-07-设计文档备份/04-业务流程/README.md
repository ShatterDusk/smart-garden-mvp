## 二、流程设计规范

### 2.1 流程文档结构

每个流程文档包含以下章节：

```
## X、流程名称
### X.1 流程概述
### X.2 流程图（Mermaid）
### X.3 步骤说明
### X.4 数据变化
### X.5 异常处理
### X.6 相关接口
```

- 使用 Mermaid sequenceDiagram 表示时序
- 使用 Mermaid flowchart 表示状态流转
- 参与者：用户、前端、后端、数据库、外部服务
