## 需求描述

在植物详情页显示健康评分的历史变化趋势图表，帮助用户了解植物健康状况的变化趋势。

### 业务场景
1. 用户进入植物详情页
2. 在诊断 Tab 查看健康评分趋势图
3. 趋势图显示最近 30 天的评分变化
4. 用户可以切换查看 7 天、30 天、90 天的趋势

---

## 实现方案

### 核心思路
**无需新建表**，直接查询 `diagnosis_cards` 表：

```sql
-- 查询植物最近30天的健康评分历史
SELECT 
  DATE(created_at) as date,
  health_score,
  status
FROM diagnosis_cards 
WHERE plant_id = ? 
  AND health_score IS NOT NULL
ORDER BY created_at DESC 
LIMIT 30;
```

### 为什么选择查询而非新建表

| 方案 | 优点 | 缺点 |
|:---|:---|:---|
| 查询 diagnosis_cards | 无数据冗余，实现简单 | 历史数据受诊断频率影响 |
| 新建 health_scores 表 | 数据独立，可控性强 | 数据冗余，需要额外维护 |

**结论**: diagnosis_cards 已经包含每次诊断的健康评分，足以支撑趋势展示需求。

## 后端实现

### 1. 添加健康趋势接口

**文件**: `server/src/controllers/plantController.js`

新增 `getHealthTrend` 函数：

```javascript
/**
 * 获取植物健康评分趋势
 * GET /api/plants/:id/health-trend
 */
async function getHealthTrend(req, res, next) {
  try {
    const { id: plantId } = req.params;
    const { days = 30 } = req.query; // 默认30天
    const { userId } = req.user;

    // 验证植物归属
    const plant = await Plant.findOne({
      where: { plant_id: plantId, user_id: userId }
    });

    if (!plant) {
      return res.error('植物不存在或无权限', 404);
    }

    // 计算起始日期
    const startDate = new Date();
    startDate.setDate(startDate.getDate() - parseInt(days));

    // 查询健康评分历史
    const diagnoses = await DiagnosisCard.findAll({
      where: {
        plant_id: plantId,
        health_score: { [Op.not]: null },
        created_at: { [Op.gte]: startDate }
      },
      attributes: ['health_score', 'status', 'created_at'],
      order: [['created_at', 'ASC']]
    });

    // 格式化趋势数据
    const trend = diagnoses.map(d => ({
      date: d.created_at.toISOString().split('T')[0],
      score: d.health_score,
      status: d.status
    }));

    // 计算统计数据
    const scores = diagnoses.map(d => d.health_score);
    const currentScore = scores.length > 0 ? scores[scores.length - 1] : null;
    const averageScore = scores.length > 0
      ? Math.round(scores.reduce((a, b) => a + b, 0) / scores.length)
      : null;
    const maxScore = scores.length > 0 ? Math.max(...scores) : null;
    const minScore = scores.length > 0 ? Math.min(...scores) : null;

    res.success({
      trend,
      currentScore,
      averageScore,
      maxScore,
      minScore,
      totalRecords: diagnoses.length
    });
  } catch (error) {
    next(error);
  }
}
```

## 前端实现

### 1. 植物详情页 - 诊断 Tab

**文件**: `pages/plant-detail/plant-detail.js`

添加趋势图数据加载：

```javascript
data: {
  // ... 其他数据
  healthTrend: [],
  trendStats: {},
  trendDays: 30, // 默认30天
},

async loadHealthTrend() {
  const { plantId, trendDays } = this.data;
  try {
    const res = await api.getPlantHealthTrend(plantId, trendDays);
    this.setData({
      healthTrend: res.data.trend,
      trendStats: {
        current: res.data.currentScore,
        average: res.data.averageScore,
        max: res.data.maxScore,
        min: res.data.minScore
      }
    });
  } catch (error) {
    console.error('加载健康趋势失败:', error);
  }
},

// 切换时间范围
switchTrendDays(e) {
  const days = e.currentTarget.dataset.days;
  this.setData({ trendDays: days }, () => {
    this.loadHealthTrend();
  });
}
```
