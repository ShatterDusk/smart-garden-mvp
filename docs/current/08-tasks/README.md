## 状态说明

- **待实现**: 已确认需求，等待开发
- **进行中**: 正在开发中
- **已完成**: 功能已实现并测试通过
- **已放弃**: 决定不实现，记录原因

## 优先级说明

- **P0**: 阻塞性需求，必须立即实现
- **P1**: 核心功能，需要尽快实现
- **P2**: 增强功能，有计划地实现
- **P3**: 优化项，有时间再实现
