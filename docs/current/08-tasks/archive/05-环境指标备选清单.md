# 环境指标备选清单

本文档列出最终选用的环境指标。

**设计原则**:
- 传感器源和天气 API 的指标编码尽量一致，便于交叉查询
- 通用指标（如温度、湿度、大气压强）同时支持多种来源，通过 `applicable_sources` 字段灵活配置
- `applicable_sources` 是 JSON 数组，支持任意组合：["sensor"], ["weather_api"], ["sensor", "weather_api"], ["sensor", "weather_api", "manual"]

**`applicable_sources` 字段用途**:
- 标识该指标可以从哪些数据来源获取
- 查询时可按来源筛选（如：只查传感器数据，或只查天气数据，或全部）
- 支持未来扩展新来源（如手动录入、第三方平台等）
- **完全灵活**：同一个指标编码可以支持多个来源，不限制来源类型

**示例场景**:
- `temperature`: ["sensor", "weather_api"] - 既有传感器数据，又有天气 API 数据，可交叉对比
- `pressure`: ["sensor", "weather_api"] - 如果传感器有气压计，也可以采集；同时天气 API 也提供
- `light_intensity`: ["sensor"] - 只有传感器能采集
- `weather_condition`: ["weather_api"] - 只有天气 API 能提供

## 一、传感器指标（已选定 7 个）

| 指标编码 | 名称 | 单位 | 测量范围 | 精度 | 图标 | 适用来源 | 优先级 |
|:---|:---|:---:|:---|:---|:---:|:---:|:---:|
| temperature | 温度 | °C | -40~85°C | ±0.5°C | 🌡️ | ["sensor", "weather_api"] | P0 |
| humidity | 湿度 | % | 0~100% | ±3% | 💧 | ["sensor", "weather_api"] | P0 |
| light_intensity | 光照强度 | lux | 0~200,000 | ±5% | ☀️ | ["sensor"] | P0 |
| soil_moisture | 土壤湿度 | % | 0~100% | ±2% | 🌱 | ["sensor"] | P0 |
| soil_temperature | 土壤温度 | °C | -20~60°C | ±0.2°C | 🌡️ | ["sensor"] | P1 |
| soil_ph | 土壤酸碱度 | pH | 3.0~9.0 | ±0.1~0.2 | 🔬 | ["sensor"] | P2 |
| battery_level | 设备电量 | % | 0~100% | - | 🔋 | ["sensor"] | P1 |

**说明**:
- **temperature/humidity**: 同时支持传感器和天气 API，可交叉对比
- **light_intensity**: 人眼感知的光照强度
- **soil_moisture**: 智能灌溉决策核心指标
- **soil_temperature**: 根系生长环境，冬季防冻
- **soil_ph**: 土壤改良指导
- **battery_level**: 设备状态监控

## 二、天气 API 指标（已选定 12 个）

根据 `/v7/weather/now` 接口返回数据选定。

### 2.1 原始 API 响应示例

```json
{
  "code": "200",
  "updateTime": "2020-06-30T22:00+08:00",
  "fxLink": "http://hfx.link/2ax1",
  "now": {
    "obsTime": "2020-06-30T21:40+08:00",
    "temp": "24",
    "feelsLike": "26",
    "icon": "101",
    "text": "多云",
    "wind360": "123",
    "windDir": "东南风",
    "windScale": "1",
    "windSpeed": "3",
    "humidity": "72",
    "precip": "0.0",
    "pressure": "1003",
    "vis": "16",
    "cloud": "10",
    "dew": "21"
  },
  "refer": {
    "sources": ["QWeather", "NMC", "ECMWF"],
    "license": ["QWeather Developers License"]
  }
}
```

### 2.2 字段映射表

| 指标编码 | 名称 | 单位 | 天气 API 字段 | 图标 | 适用来源 | 优先级 |
|:---|:---|:---:|:---|:---:|:---:|:---:|
| temperature | 温度 | °C | temp | 🌡️ | ["weather_api"] | P0 |
| humidity | 湿度 | % | humidity | 💧 | ["weather_api"] | P0 |
| feels_like | 体感温度 | °C | feelsLike | 🌡️ | ["weather_api"] | P2 |
| weather_condition | 天气状况 | - | text | ☀️ | ["weather_api"] | P0 |
| wind_direction | 风向 | - | windDir | 🧭 | ["weather_api"] | P1 |
| wind_direction_360 | 风向角度 | ° | wind360 | 🧭 | ["weather_api"] | P2 |
| wind_scale | 风力等级 | 级 | windScale | 💨 | ["weather_api"] | P1 |
| wind_speed | 风速 | km/h | windSpeed | 💨 | ["weather_api"] | P1 |
| precip | 降水量 | mm | precip | 🌧️ | ["weather_api"] | P1 |
| pressure | 大气压强 | hPa | pressure | 🌐 | ["sensor", "weather_api"] | P2 |
| visibility | 能见度 | km | vis | 👁️ | ["weather_api"] | P2 |
| cloud_cover | 云量 | % | cloud | ☁️ | ["weather_api"] | P3 |
| dew_point | 露点温度 | °C | dew | 💧 | ["weather_api"] | P3 |

### 2.3 字段说明

| API 字段 | 指标编码 | 说明 | 示例值 |
|:---|:---|:---|:---|
| temp | temperature | 温度，默认单位：摄氏度 | "24" |
| humidity | humidity | 相对湿度，百分比数值 | "72" |
| feelsLike | feels_like | 体感温度，默认单位：摄氏度 | "26" |
| text | weather_condition | 天气状况的文字描述 | "多云" |
| icon | - | 天气状况的图标代码（存储在 weather_condition 中） | "101" |
| windDir | wind_direction | 风向 | "东南风" |
| wind360 | wind_direction_360 | 风向360角度 | "123" |
| windScale | wind_scale | 风力等级 | "1" |
| windSpeed | wind_speed | 风速，公里/小时 | "3" |
| precip | precip | 过去1小时降水量，默认单位：毫米 | "0.0" |
| pressure | pressure | 大气压强，默认单位：百帕 | "1003" |
| vis | visibility | 能见度，默认单位：公里 | "16" |
| cloud | cloud_cover | 云量，百分比数值 | "10" |
| dew | dew_point | 露点温度 | "21" |

**注意**: 
- `icon` 字段是天气图标的代码，已合并到 `weather_condition` 中存储，不作为独立指标
- `obsTime` 是数据观测时间，用于数据时效性校验
- `updateTime` 是 API 更新时间，用于缓存策略
