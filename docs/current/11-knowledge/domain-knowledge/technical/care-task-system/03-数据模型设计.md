## 2. CareTask 模型（养护任务）

### 2.1 数据库表结构

```sql
CREATE TABLE care_tasks (
  -- 主键
  id INT PRIMARY KEY AUTO_INCREMENT,
  task_id VARCHAR(64) UNIQUE NOT NULL COMMENT '任务编号 TASK_xxx',
  
  -- 关联信息
  plant_id VARCHAR(64) NOT NULL COMMENT '植物ID',
  user_id VARCHAR(64) NOT NULL COMMENT '用户ID',
  diagnosis_card_id VARCHAR(64) COMMENT '关联诊断卡ID',
  
  -- 任务内容
  task_type VARCHAR(32) NOT NULL COMMENT '任务类型: water/fertilize/prune/repot/pestControl/other/urgent_care',
  title VARCHAR(128) NOT NULL COMMENT '任务标题',
  description TEXT COMMENT '任务描述',
  
  -- 时间信息
  scheduled_date DATE NOT NULL COMMENT '计划执行日期',
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  completed_at DATETIME COMMENT '完成时间',
  
  -- 来源信息
  source VARCHAR(32) NOT NULL COMMENT '来源: ai/rule/manual',
  ai_reason TEXT COMMENT 'AI推荐理由（AI生成时填写）',
  
  -- 状态
  status VARCHAR(32) DEFAULT 'pending' COMMENT '状态: pending/completed/skipped/expired',
  priority VARCHAR(32) DEFAULT 'normal' COMMENT '优先级: high/normal/low',
  
  -- 执行信息（完成后填充）
  execution_description TEXT COMMENT '执行描述',
  execution_images JSON COMMENT '执行图片数组',
  care_record_id VARCHAR(64) COMMENT '关联的养护记录ID',
  
  -- 索引
  INDEX idx_plant_id (plant_id),
  INDEX idx_user_id (user_id),
  INDEX idx_status (status),
  INDEX idx_scheduled_date (scheduled_date),
  INDEX idx_source (source),
  INDEX idx_diagnosis_card (diagnosis_card_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='养护任务表';
```

### 2.2 Sequelize 模型定义

```javascript
// backend/server/src/models/CareTask.js
const { DataTypes } = require('sequelize');

    type: DataTypes.STRING(64),

    type: DataTypes.STRING(64),

    type: DataTypes.TEXT,
```

## 3. CareRecord 模型（养护记录）

### 3.1 扩展现有模型

基于现有 `CareRecord` 模型，增加 `source` 和 `taskId` 字段：

```javascript
// backend/server/src/models/CareRecord.js
// 在现有基础上增加字段

module.exports = (sequelize, DataTypes) => {
  const CareRecord = sequelize.define('CareRecord', {
    // ... 现有字段 ...
    
    // 新增字段
    source: {
      type: DataTypes.ENUM('manual', 'task'),
      defaultValue: 'manual',
      comment: '来源: manual-手动记录, task-任务完成'
    },
    taskId: {
      type: DataTypes.STRING(64),
      allowNull: true,
      field: 'task_id',
      comment: '关联的任务ID（任务完成时填写）'
    }
  }, {
    tableName: 'care_records',
    timestamps: true
  });
  
  return CareRecord;
};
```

## 5. API 数据格式

```javascript

{
  "plantId": "PLANT_001",
  "taskType": "water",
  "title": "浇水",
  "description": "土壤干燥，建议浇水",
  "scheduledDate": "2026-04-12",
  "priority": "high",
  "source": "manual"
}

// PUT /api/care-tasks/:taskId/complete - 完成任务
{
  "description": "已完成浇水，浇透",

}
```
