## 一、工作区概述

本工作区用于对 PlantGPT 项目进行**系统性静态代码分析**，识别潜在问题、安全风险和代码质量问题。

### 分析范围
- **前端**: 微信小程序（27个JS文件）
- **后端**: Node.js/Express（80+个文件）
- **数据库**: Sequelize模型（13个模型）
- **测试**: Jest测试套件
