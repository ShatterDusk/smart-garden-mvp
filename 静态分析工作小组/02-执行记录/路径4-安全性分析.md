**修复时间**: 2026-04-08  
**修复文件**: `backend/server/src/middleware/auth.js:L11-15`

**修复前**:
```javascript
const JWT_SECRET = process.env.JWT_SECRET || 'your-secret-key';
```

**修复后**:
```javascript
// JWT 密钥必须从环境变量读取，禁止硬编码
const JWT_SECRET = process.env.JWT_SECRET;
if (!JWT_SECRET) {
  throw new Error('JWT_SECRET 环境变量必须设置，用于 JWT 签名和验证');
}
```

**验证方式**: 启动服务时不设置 `JWT_SECRET`，应报错并退出


**修复后**:
```javascript
// SSL 证书验证配置：
// - 默认启用验证（安全）
// - 可通过环境变量 SSL_VERIFY=false 在特殊环境禁用

    rejectUnauthorized: process.env.SSL_VERIFY !== 'false',
  }),
});
```

**验证方式**: 默认情况下 SSL 验证启用，可通过 `SSL_VERIFY=false` 临时禁用
